# [위클리 챌린지] 1주차

## 📋 과제

1. CLI(Command Line Interface) 프로그램 제작
    - 콘솔 형태의 프로그램은 어떠한 종류라도 상관 없다.
    - 파이썬의 input() 함수와 print() 함수를 사용하여 작성한다(input()은 선택)

## 📢 설명

사용자의 일정을 기반으로 '오늘의 문장'을 출력하는 CLI 프로그램 'DailyMate'를 구현했습니다.

| 기능 | 설명 |
|---|---|
| 기능 | 1. 사용자는 자신의 하루 계획이나 일정을 자연어 문장 형태로 입력<br>2. 사용자가 입력한 문장 속 특정 키워드를 추출해 사용자 상황에 맞는 '운세/루틴/글귀'를 추천<br>3. 사용자는 반복 or 종료 키워드를 통해 프로그램 진행 |
| 추가<br>목표 | 1. 현재는 mock 데이터를 기반으로 키워드 및 '오늘의 문장(운세/루틴/글귀)' 데이터를 관리하고 있으나, 추후에는 LLM 기반 AI 모델을 활용해 사용자의 입력 상황에 맞는 문장을 생성할 수 있도록 확장<br>2. 사용자별 입력 문장과 대화 이력을 저장/분석하는 기능을 추가해, 개인의 관심사와 사용 패턴을 반영한 맞춤형 챗봇 시스템으로 확장 |

## 🚀 사용법

```bash
python daily_mate.py --name 사용자명
```

| 번호 | 기능 |
|---|---|
| 1 | 운세 추천 |
| 2 | 루틴 추천 |
| 3 | 글귀 추천 |

## 💻 실행 예시

```bash
python daily_mate.py --name emjay
=====================================
Bot: 안녕하세요, emjay님!
Bot: 오늘. 당신의 계획을 말해주세요.

You: 오늘 야구 보러 경기장 갈 거야

Bot: 무엇을 도와드릴까요?
=========================
|   번호   |    항목    |
=========================
|    1     |    운세    |
|    2     |    루틴    |
|    3     |    글귀    |
=========================

You: 운세
You: 오늘의 운세를 알려줘

Bot: 오늘은 결과보다 과정을 즐기면 더 좋은 하루가 될 거예요.

Bot: 더 확인하고 싶은 것이 있나요?

You: 아니

Bot: 종료하겠습니다.
```

## 📝 회고
- 주제 선정
1주차 과제를 시작하면서 주제 선정을 가장 고민했습니다. 추후 AI 기능을 접목하여 확장할 수 있는 구조의 프로그램을 설계하고자 했습니다. 사용자의 자연어 입력을 기반으로 키워드를 추출하고, 이에 맞는 결과를 추천하는 구조를 구현하면 이후 LLM 기반 AI 서비스나 개인화 추천 시스템으로 확장할 수 있을 것이라 판단했습니다. 이에 사용자의 하루 일정을 입력받아 운세/루틴/글귀를 추천하는 'DailyMate' 프로그램을 주제로 선정했습니다.

- 과제 수행
    - argparse 라이브러리를 학습했습니다. argparse를 활용하면 명령 인자를 받아 프로그램을 보다 유연하게 실행할 수 있음을 이해했습니다. 특히 CLI 환경에서는 반복 작업을 자동화하고 생산성을 높이는 것이 중요하다는 점을 학습하고, 이를 과제에 적용해보며 실제 사용 흐름을 경험했습니다.
    - Json 파일을 활용해 키워드와 추천 데이터를 코드와 분리했습니다. 이를 통해 유지보수와 AI 모델 연동 및 확장성을 고려하고자 했습니다.
